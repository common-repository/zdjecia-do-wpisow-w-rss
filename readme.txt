Wtyczka dodająca zdjęcia wyróżniające do każdego wpisu w kanale RSS.

== Description ==

Jeżeli Twoje wpisy w kanale RSS nie posiadają zdjęć wyróżniających, zainstaluj naszą wtyczkę. Dodaje ona do każdego wpisu w kanale RSS jego zdjęcie wyróżniające w rozmiarze "LARGE".

== Installation ==

Instalacja z pliku ZIP:

1. Zaloguj się do panelu administracyjnego Twojego bloga.
2. Z bocznego menu wybierz opcję "Wtyczki", a następnie "Dodaj nową".
3. Kliknij opcję "Wyślij wtyczkę na serwer".
4. Wybierz plik "ZIP" z naszą wtyczką i kliknij "Zainstaluj teraz".
5. Włącz wtyczkę.

Instalacja z rozpakowanego pliku ZIP:

1. Umieść rozpakowany folder na serwerze w katalogu wtyczek Twojego bloga (wp-content/plugins).
2. Zaloguj się do panelu administracyjnego Twojego bloga.
3. Z bocznego menu wybierz opcję "Wtyczki".
4. Włącz wtyczkę "Zdjęcia do wpisów w RSS".

Możesz również zainstalować wtyczkę prosto z katalogu wtyczek systemu WordPress.
